Скрипт оставляет бота на текущем домене.

1. Поднимаем CPA трекер и создаем оффер(ссылку), которая будет вести обратно на наш домен со скриптом, но при этом будет передавать GET запрос. Если этот гет запрос соответствует ожиданию скрипта, то оставляем пользователя или бота на текущей странице.

Если же CPA трекер провел юзера/бота по всем фильтрам и отпустил, то ведет на офферную ссылку.

Таким образом Мы не делаем редирект на другой белый сайт и не вызываем наичием редиректа подозрений инстаграма.


Для корректной работы нужно 2 ссылки на трекере. Одна просто пропускает весь прямой траф с инсты, вторая фильтрует по всем критериям всех, кто пришел не с инсты и пропускает только через фильтр. Так-же можно просто перекрыть траф через фильтр вообще и всех возвращать домой. Таким оразом пропуская траф только с инстаграма и при этом отслеживать всех остальных
